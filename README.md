# Roof Generator

Design by Mas Fikri

## Getting Started

### Prerequisites

This project uses venv to install the requirement <br/>
For Windows:

```shell
$ python -m venv venv
```

For Mac Os:

```shell
$ .venv/bin/activate
```

<!-- venv activation -->

### Venv Activation

This project uses venv to install the requirement <br/>
For Windows:

```shell
$ venv/Scripts/activate
```

For Mac Os:

```shell
$ source venv/bin/activate
```

<!-- Installation -->

### Installation

Install all requirements in venv

```shell
(venv) $ pip install -r requirements.txt
```

<!-- Run Local -->

### Run Local

```shell
(venv) $ python main.py
```
